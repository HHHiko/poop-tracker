## 漫画风设计语言

### 核心特征

1. **粗黑描边**：所有卡片、按钮、输入框使用 `3rpx solid #2C2C2C` 边框
2. **偏移阴影**：`3-4rpx 3-4rpx 0 #2C2C2C` 实色投影（漫画对话框风格）
3. **按下反馈**：`:active` 时阴影缩小 + `translate` 位移，营造"按下去"的物理感
4. **像素方块**：标记点使用 `border-radius: 2-4rpx` 小方块代替圆点
5. **圆角适度**：卡片 `12rpx`，按钮 `12rpx`，不要太圆以保持漫画感

## 字体规范

| 层级 | 大小 (rpx) | 用途 |
|------|-----------|------|
| H1 | 40rpx | 页面标题、月份文字 |
| H2 | 32rpx | 卡片标题、日期数字 |
| Body | 28rpx | 正文、记录详情 |
| Caption | 24rpx | 辅助说明、备注 |
| Small | 20rpx | 日历格子小标记文字 |

字体：系统默认（微信小程序会自动使用系统字体），加粗 `font-weight: bold` 更符合漫画风格。

## 组件规范

### 卡片

- 背景：`#FFFFFF`
- 边框：`3rpx solid #2C2C2C`
- 圆角：`12rpx`
- 阴影：`4rpx 4rpx 0 #2C2C2C`（漫画偏移阴影）
- 内边距：`24rpx`

### 主按钮

- 背景：`#42A5F5`
- 文字：`#FFFFFF`
- 边框：`3rpx solid #2C2C2C`
- 圆角：`12rpx`
- 高度：`88rpx`
- 字号：`32rpx`
- 阴影：`4rpx 4rpx 0 #2C2C2C`
- 按下态：底色变深 + 阴影缩小 + 位移 `translate(2-3rpx)`

### 便便类型选择卡片

- 未选中：白色背景 + `3rpx solid #2C2C2C` 黑边 + 偏移阴影
- 选中态：类型对应颜色背景 + 同色边框 + 文字变白 + emoji 放大
- 3×2 网格排列（第5个卡片单独一行）
- 每个卡片包含：大图标(emoji 52rpx) + 类型名称(28rpx) + 说明(22rpx)

### FAB 浮动按钮

- 圆角方形 `16rpx`，尺寸 `112rpx × 112rpx`
- 背景：`#42A5F5`
- 图标：白色 "+"，大小 `56rpx`
- 边框：`3rpx solid #2C2C2C`
- 阴影：`4rpx 4rpx 0 #2C2C2C`
- 位置：右下角，距底部 `120rpx`，距右侧 `48rpx`
- 按下态：位移加深色

### 输入框

- 背景：`#FFFFFF`
- 边框：`3rpx solid #2C2C2C`
- 圆角：`12rpx`
- 阴影：`3rpx 3rpx 0 #2C2C2C`
- 内边距：`20rpx 24rpx`
- 字号：`28rpx`
