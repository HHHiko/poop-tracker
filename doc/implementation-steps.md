## 阶段 1：项目骨架搭建

### 目标
创建微信小程序项目的最小可运行骨架。

### 步骤

1. **创建 `miniprogram/` 目录结构**
   - `miniprogram/app.js` — 小程序入口
   - `miniprogram/app.json` — 全局配置
   - `miniprogram/app.wxss` — 全局样式
   - `miniprogram/utils/storage.js` — 数据工具（占位）
   - `miniprogram/pages/calendar/` — 4 个文件
   - `miniprogram/pages/record/` — 4 个文件
   - `miniprogram/pages/detail/` — 4 个文件

2. **配置 app.json**
   - `pages` 路由注册
   - `tabBar` 配置（月历 + 记录，蓝色主题）
   - `window` 全局窗口样式

3. **编写 app.wxss 全局样式**
   - 页面背景色 `#FFF8E1`
   - 通用卡片样式 `.card`
   - 通用按钮样式 `.btn-primary`
   - 配色 CSS 变量（或常量注释）

4. **每个页面放占位内容**
   - 显示页面名称，确认路由和 tabBar 正常切换

### 验收标准
- [ ] 在微信开发者工具中打开项目，无报错
- [ ] 底部 tabBar 显示两项：月历、记录
- [ ] 点击 tabBar 可切换页面
- [ ] 全局浅黄色背景生效
